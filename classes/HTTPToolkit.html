    requestHead

    • requestHead(__namedParameters: Request): Promise<HeadResponse>

      Sends a HEAD request, falling back to simulation when the server does not support HEAD. Tries, in order:

      1. A standard HEAD request via fetch().
      2. A Range GET to retrieve the first FILE_TYPE_MAGIC_NUMBER_SIZE bytes (magic-number file header).
      3. A plain GET that reads the first FILE_TYPE_MAGIC_NUMBER_SIZE bytes then cancels the body stream.
